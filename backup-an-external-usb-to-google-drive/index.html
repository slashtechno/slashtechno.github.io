If you use Ubuntu (or even other Linux Distros), Deja Dup can be used to backup storage drives to many different storage locations, including Google Drive.

Installation

You can easily install Deja Dup with Snap. After installing Snap, you can run:
sudo snap install deja-dup --classic

Usage

Just open up Deja Dup, select your backup location, what to backup, and begin the backup. The interface is quite straightforward and easy to use.

One more thing

If you are using Ubuntu, you may want to uninstall Deja Dup if it is installed. This is because the preinstalled version of Deja Dup may not be using Snap.
